---
title: "run"
description: "Executa o workflow usando a versão especificada e as variáveis"
---



Se nenhuma versão for fornecida, a versão mais recente é usada.

```python
workflow = notte.Workflow("<your-workflow-id>")
workflow.run(variable1="value1", variable2="value2")
```

> Certifique-se de que as variáveis corretas sejam fornecidas com base no arquivo Python enviado anteriormente. Caso contrário, o workflow falhará.

Você pode usar `local=True` para executar o workflow localmente se tiver acesso à chave de descriptografia
