---
title: "bifurcar"
description: "Bifurque um workflow compartilhado em seu próprio workflow privado"
---



```python
workflow = notte.Workflow("<user-shared-workflow-id>")
forked_workflow = workflow.fork()
forked_workflow.run()
```

O workflow bifurcado é acessível apenas para você e você pode atualizá-lo como quiser
