---
title: "atualizar"
description: "Atualize o workflow com uma nova versão do código"
---



```python
workflow = notte.Workflow("<your-workflow-id>")
workflow.update(workflow_path="<path-to-your-workflow.py>")
```

Se você definir uma versão, apenas essa versão será atualizada


## Parâmetros

<ParamField path="workflow_path" type="str" required>
</ParamField>

<ParamField path="version" type="UnionType[str, None]" default="None">
</ParamField>

## Retorna

`None`
